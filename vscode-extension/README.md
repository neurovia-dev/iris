# IRIS Language Support for VS Code

Syntax highlighting and snippets for the IRIS DSL.

## Features

- Syntax highlighting for `.iris` files
- Code snippets for common patterns
- Comment support (`//` and `/* */`)
- Auto-closing brackets

## Installation

### From Source

1. Copy this folder to your VS Code extensions directory:
   - **Windows**: `%USERPROFILE%\.vscode\extensions\`
   - **macOS**: `~/.vscode/extensions/`
   - **Linux**: `~/.vscode/extensions/`

2. Reload VS Code

### Development

1. Open this folder in VS Code
2. Press `F5` to launch a new VS Code window with the extension loaded
3. Open a `.iris` file to test

## Snippets

- `V` - VStack
- `H` - HStack
- `S` - Skeleton
- `x` - Repeat
- `Vg` - VStack with gap
- `Hg` - HStack with gap
- `Swh` - Skeleton with width/height
- `Sr` - Skeleton rounded
- `Vpm` - VStack with padding/margin
- `card` - Card pattern

## Language Features

- Syntax highlighting for keywords (V, H, S, x)
- Parameter highlighting (g, p, m, b, etc.)
- Number highlighting
- Comment support
